# Lokale Trainingsbilder

Dieses Verzeichnis ist für lokale Entwicklungs- und Erkennungstests vorgesehen. Bilddateien werden durch die `.gitignore` in diesem Ordner nicht versioniert, da sie Kennzeichen oder andere personenbezogene Daten enthalten können.

Für öffentlich versionierte Tests ausschließlich synthetische oder zuverlässig anonymisierte Bilder verwenden.
